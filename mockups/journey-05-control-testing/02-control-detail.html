Logical and Physical Access Controls

CTRL-001 In Progress
Control Description
Description
The entity implements logical access security software, infrastructure, and architectures over protected information assets to protect them from security events. Physical access to facilities, protected information assets, and production systems is restricted to authorized personnel.
Control Objective
To ensure that logical and physical access to information assets is restricted to authorized individuals through appropriate access control mechanisms, authentication methods, and physical security measures.
